# Dr. Vivek Karn

Pediatric resident at TU Teaching Hospital, Nepal.
Full-time doctor, part-time coder.

I build tools for problems I run into during ward rounds, night duties, NICU shifts, and exam prep. Most are designed for low-resource settings where internet access is unreliable and good clinical references are difficult to access quickly.
