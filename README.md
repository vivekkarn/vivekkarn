# Dr. Vivek Karn

Pediatric resident. Building AI tools for clinicians and students at night.

I run into problems at the hospital and write code to fix them —
exam prep, offline clinical references, evidence search, syndrome lookup.
Most of my tools are built for low-resource settings where internet is unreliable
and good references are expensive.

## Projects

### [PediaHelper.com](https://pediahelper.com)
Pediatric learning and clinical support platform.

| Tool | What it does |
|------|-------------|
| [Ask Nelson](https://pediahelper.com/ask-nelson) | Rapid AI-assisted revision grounded in Nelson Textbook concepts |
| [Ask Evidence](https://pediahelper.com/ask-evidence) | PubMed-backed literature search with AI-synthesized clinical summaries and real PMIDs |

### [PocketMed AI](https://play.google.com/store/apps/details?id=com.vivek.pocketmed) `Android`
Offline-first clinical assistant. No internet required.

- On-device AI for drug dosing, differentials, lab interpretation, and acute management
- Built for real hospital workflows in Nepal and similar low-resource settings
- Privacy-first architecture — nothing leaves the device
